# ☕ Cafe-Expresso

Sistema de autoatendimento para cafeteria desenvolvido em Java.

---

## 📌 Descrição

O "Café Expresso" é um sistema que permite ao cliente montar seu próprio pedido em um totem digital e realizar o pagamento no mesmo local, sem filas.

---

## ⚙️ Funcionalidades

- Cadastro de produtos
- Criação de pedidos
- Cálculo automático do valor total
- Alteração do status do pedido

---

## 💻 Tecnologias

- Java
- Programação Orientada a Objetos (POO)
